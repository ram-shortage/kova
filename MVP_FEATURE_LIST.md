# MVP Feature List

## Must-have
1. Brand setup
   - Upload logo(s), colors, and fonts.
   - Define font roles (title, subtitle, body).
2. Style controls
   - Style families and mood presets.
   - Basic controls for spacing and typography scale.
3. Layout library
   - 8-12 core slide types:
     - Title, section, agenda, content, media, comparison, timeline, quote, data.
4. Preview system
   - Live preview of the template set.
   - Generate 3 variants per style change.
5. Export
   - PPTX export with editable master slides.
   - Keynote export.
   - Google Slides export.
6. Quality checks
   - Contrast and spacing checks.
   - Brand compliance warnings.

## Should-have
- Style tokens editor (advanced adjustments).
- Icon set selector and accent motif system.
- Save and reuse template presets.

## Nice-to-have
- Marketplace-ready template pack export format.
- Export preset profiles for different use cases (pitch, report, training).
